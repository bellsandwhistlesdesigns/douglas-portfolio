## About This Project

This portfolio was built to reflect both the technical and design sides of my work. It highlights:

- AV system design and large-scale deployment experience  
- Network infrastructure and engineering projects  
- Full-stack development capabilities  
- Real-world case studies from commercial environments  
- A clean, responsive, and mobile-first design approach  

The goal was to create a site that feels **fast, modern, and easy to navigate**, while still communicating technical depth.
